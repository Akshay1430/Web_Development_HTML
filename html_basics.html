* br Tag

br tag is use for adding space between two lines.This tag is self closing tag hence doesn't need a closing tag like other tags syntax is < br >

* hr Tag

Hr tag is use to add horizontal lines in the webpage. To elaborate this more effectively we need to understand HTML Attributes. attributes specifies additional requirements regarding a particular tag. e.g. if we need to decide color alignment size shading etc of the line which we are about to add in the webpage, we will use html attributes. syntax is < hr color = " #080808 " size=" 3 " noshade="" > < /hr >


* Center Tag

Center tag is use to align the text/headline in the center position of the page.

* HTML Boilerplate

Just like there is structure to write a letter or email where you have to put certain details like address, greetings, main messege and signature at certain place in the letter, the same way HTML also has a structure, which is known as HTML Boilerplate.

  • < ! Doctype html > -- This statement defines the Doctype of file for browser to read, here it is HTML file.

  • < html lang = "en" > -- Latest verison of html is defined here by < html tag and set english as language. all the code will be written inside this tag

  • < head > -- this element doesn't show any content on the website but contains important information about the website

  • < meta charset="UTF-8" > -- this is part of head element which insures that text on the website displays correctly

  • < title > HTML BASICS < / title > -- this is title which displays in the link of the webpage

  • < / head > -- closing of the head element after giving all the important information to the browser for smooth rendering of the webpage

  • < body > -- inside body element we will write our content which needs to be displayed on the webpage to our users

  • < h1> Hello World -- content with headline 1 style< / h1 >

  • < / body> -- body element closing tag

  • < / html > -- html closing tag

This is a HTML Boilerplate. Shortcut for this Boilerplate is ! + enter button. It will automaticaly write everything for you.

Table

When we want to put data in a structural tabular format table tag is used.There are many attributes to design or style the table like align or border or bgcolor. But avoid using such attributes as we will use css for styling of table.
  • thead is table head
  • tfoot is table foot
  • tr is table row - data row
  • td is table data - table content
  • th is table header - column name
  • tbody is table body - where row and data is mentioned
  • cellspacing is used to define space between row element

    • Always mention Table head inside thead tag,Mention Table containt inside tbody tag and mention Table foot inside tfoot tag.

    tr and th tags are used inside thead tag. while tr and td tags are elements of tbody tag

HTML Forms

HTML Forms are used to take inputs from users. Input type contains Text, Color, Submit(to create submit button),, Checkbox(to create check box), etc.

Syntax is

    < form class="" action="HTML File Name" method="post" >
    < label > Your Name < /label >
    < input type="text" name="" value="" >
    < input type="Submit" name="" value="" >
    < input type="file" name="" value="" >
    < input type="date" name="" value="" >
    < input type="password" name="" value="" >
    < input type="search" name="" value="" >
    < input type="number" name="" value="" >
    < /form >
